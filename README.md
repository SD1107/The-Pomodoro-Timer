# ⏰ Pomodoro Timer

A simple and aesthetic **Pomodoro Timer** built using Python and Tkinter. This desktop app helps you stay productive by using the proven **Pomodoro technique**: 25 minutes of focused work followed by short and long breaks.

![UI Screenshot](tomato.png) <!-- Optional: Replace with actual screenshot of the app -->

---

## 🧠 Features

- ✅ 25-minute **work** sessions
- 💤 5-minute **short breaks**
- 🌴 30-minute **long breaks** after every 4 work sessions
- 🔁 Automatic session transitions
- 🔃 **Reset** button to restart the timer anytime
- ✔️ Progress checkmarks to track completed Pomodoros
- 🎨 Stylish and colorful UI using Tkinter

---

## 📦 Requirements

- Python 3.x
- Tkinter (usually included by default with Python)
- `tomato.png` image (placed in the same directory)
